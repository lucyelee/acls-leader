# ACLS Team Leader 📱

심폐소생술(CPR) 코드 상황에서 팀 리더를 보조하는 실시간 대시보드입니다.

iPhone/Android 홈 화면에 추가하면 오프라인에서도 독립 앱처럼 동작합니다.

## 주요 기능

- ⏱ **마스터 타이머** — 코드 경과 시간 실시간 표시
- 💓 **CPR 관리** — 1분 교대 타이머 + 2분마다 리듬체크 안내 + 110 BPM 메트로놈
- 🫁 **Airway** — BVM (30:2) / LMA / ETT 전환
- ⚡ **Rhythm** — VF, pVT, PEA, Asystole, ROSC 선택 + 제세동 카운트
- 💊 **Drugs** — Epinephrine (q3-5분 타이머), Amiodarone (300→150mg)
- ⚠️ **H5T4** — 가역적 원인 체크리스트
- 📋 **Event Log** — 모든 이벤트 자동 기록 + 텍스트 내보내기

## 설치 방법

### iPhone (Safari)
1. Safari에서 이 페이지 URL 접속
2. 하단 공유 버튼 (□↑) 탭
3. **"홈 화면에 추가"**
4. 완료 — 오프라인으로 작동합니다

### Android (Chrome)
1. Chrome에서 이 페이지 URL 접속
2. 메뉴 (⋮) → **"앱 설치"**

## 기술 스택
