# 2025 ACLS Team Leader Dashboard 📱

심폐소생술(CPR) 코드 상황에서 팀 리더를 보조하는 실시간 대시보드입니다.
iPhone/Android 홈 화면에 추가하면 오프라인에서도 독립 앱처럼 동작합니다.

A real-time dashboard to assist team leaders during cardiac arrest (CPR code) situations.
Add to your iPhone/Android home screen and it works offline as a standalone app.

🌐 **[한국어](index.html)** · **[English](en.html)**

---

## Features / 주요 기능

- ⏱ **Master Timer** — Real-time code elapsed time / 코드 경과 시간 실시간 표시
- 💓 **CPR Management** — 1-min swap timer + rhythm check every 2 cycles + 110 BPM metronome
- 🫁 **Airway** — BVM (30:2) / LMA / ETT switching
- ⚡ **Rhythm** — VF, pVT, PEA, Asystole, ROSC + defibrillation counter
- 💊 **Drugs** — Epinephrine (q3-5min timer), Amiodarone (300→150mg) with smart dosing guide
- 🔥 **Refractory VF** — DSED pad placement guide with anatomical diagrams
- ⚠️ **H5T4** — Reversible causes checklist / 가역적 원인 체크리스트
- ✅ **ROSC** — Post-cardiac arrest care checklist
- 📋 **Event Log** — Auto-records all events + text export for debriefing

## Install / 설치 방법

### iPhone (Safari)
1. Open the URL in Safari
2. Tap the share button (□↑)
3. **"Add to Home Screen"**
4. Done — works offline!

### Android (Chrome)
1. Open the URL in Chrome
2. Menu (⋮) → **"Install app"**

Single HTML file + PWA (Service Worker). Zero framework dependencies.
